## 使用场景

展示行列数据。

### 代码展示

##### 使用方法

```js
$("绑定的元素").jlistMenu({ 需传递的参数 });
```

##依赖项

[list 列表](http://docs/jui-jlist-cn)

## 参数

| 参数     | 类型   | 是否必须 | 默认值 | 注释                      |
| -------- | ------ | -------- | ------ | ------------------------- |
| itemTmpl | String | 否       | 无     | 菜单 DOM 元素             |
| mode     | String | 否       | `'v'`  | 展示方法/v：一列，h：两列 |
