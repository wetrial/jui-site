# wetrial Site

> [参考地址](https://github.com/ant-design/ant-design-pro-site)

[jui](https://github.com/wetrial/jui-site)的使用文档.

### 开发

```bash
$ git clone git@github.com:wetrial/jui-site.git
$ cd jui-site
$ npm install
$ npm start
```

Then visit http://localhost:8001 .

### 部署

```
$ npm run site
```
